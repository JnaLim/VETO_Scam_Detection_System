# Project Docs

This folder is the project memory for AI-assisted development work in this repository. Keep it current when work changes project state, decisions, bugs, handover context, or workflow assumptions.

## Files

- `status.md` records current project status, branch state, goals, and known risks.
- `progress.md` records dated progress entries after meaningful work.
- `handover.md` records unfinished work, next steps, and context for the next thread.
- `decisions.md` records important decisions and the reasoning behind them.
- `bugs.md` records bug investigations, root causes, fixes, and verification.

Keep entries short, factual, and useful to the next agent.
